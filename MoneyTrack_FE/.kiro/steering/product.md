# Product: MoneyTrack FE

A personal finance manager web app (frontend only) that lets users track income, expenses, and debts. The UI is in Vietnamese.

Core features:
- Dashboard overview with income/expense summary and pie chart by category
- Transaction ledger grouped by date, filterable by category
- Hierarchical category system (parent/child) for income and expenses
- Transaction creation via a floating action button
- Admin section for user management
- Account page for user profile

The app currently uses mock data (`lib/mock-data.ts`) — there is no real backend or API integration yet.
