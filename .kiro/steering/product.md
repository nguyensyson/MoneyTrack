# MoneyTrack – Product Overview

MoneyTrack is a personal finance tracking application that lets users record, categorize, and analyze their income and expenses.

## Core Features
- User authentication (register/login) with role-based access (USER, ADMIN)
- Transaction management: create, view, and filter income/expense transactions
- Category management: hierarchical categories (parent/child) with INCOME or EXPENSE type
- Statistics dashboard: monthly summaries, balance, and expense breakdown by category
- Admin panel: manage categories and view system-wide data

## User Roles
- `USER` – manages their own transactions and views personal statistics
- `ADMIN` – manages categories and accesses the admin dashboard
